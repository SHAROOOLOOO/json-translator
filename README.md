# JSON翻译工具

专业的JSON数据翻译Web应用，支持批量翻译JSON结构中的文本字段。

## ✨ 核心功能

- **Monaco编辑器**：VS Code级别的编辑体验，语法高亮和格式化
- **实时解析**：自动解析JSON结构，提取可翻译字段
- **批量翻译**：支持选择特定字段进行批量翻译
- **多语言支持**：支持中文、英文、日文、韩文等9种语言
- **翻译结果编辑**：翻译完成后可实时编辑和调整

## 🌐 翻译服务

### 翻译策略
1. MyMemory API (每日免费10,000次)
2. Google Translate (高质量翻译)
3. LibreTranslate (开源服务)
4. 本地词典 (备用方案)

### 支持语言
- 中文、英文、日文、韩文、法文、德文、西班牙文、俄文、阿拉伯文

## 🚀 快速开始

```bash
# 安装
npm install

# 启动开发
npm run dev
# 访问: http://localhost:5173

# 构建
npm run build
```

## 📖 使用流程

1. **输入JSON**：在左侧编辑器输入JSON数据
2. **选择字段**：系统自动解析，勾选需要翻译的字段
3. **选择语言**：选择目标语言，点击翻译
4. **编辑结果**：在右侧编辑器查看和编辑翻译结果

## 🛠️ 技术栈

- **前端**：React 19 + TypeScript
- **构建**：Vite 7
- **样式**：Tailwind CSS
- **编辑器**：Monaco Editor

## 📝 示例

**输入**:
```json
{
  "name": "张三",
  "city": "北京"
}
```

**输出**:
```json
{
  "name": "Zhang San",
  "city": "Beijing"
}
```

## ⚠️ 注意事项

- 需要网络连接进行翻译
- 翻译数据会发送到第三方服务
- 免费翻译服务有使用限制
